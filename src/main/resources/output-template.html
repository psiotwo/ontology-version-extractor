Latest OBO ontology versions

See the OBO Foundry Identifier Policy for details on correct version identification in OBO Foundry. The following version types are recognized:
  • STANDARD_OBO_VERSION_IRI_1 = owl:versionIri compliant with the OBO Foundry Identifier Policy.
  • STANDARD_OBO_VERSION_IRI_2 = owl:versionIri compliant with the OBO Foundry Identifier Policy.
  • NON_STANDARD_OBO_VERSION_IRI = owl:versionIri present, with partial compliance with the OBO Foundry Identifier Policy.
  • NON_OBO_VERSION_IRI = owl:versionIri successfully extracted from owl:versionIri, but not OBO compliant.
  • NO_VERSION_IRI_BUT_VERSIONINFO = No/invalid owl:versionIri present, only owl:versionInfo.
  • NO_VERSION_INFORMATION = No owl:versionIri neither owl:versionInfo present.
  • UNKNOWN = A problem (e.g. failing connection) occurred when fetching ontology header.
